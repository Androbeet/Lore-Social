# 🗺 LORE — Roadmap

What's live, what's next, and what each phase needs. All phases stay at $0.

## ✅ Phase 0 — SHIPPED (this repo, demo mode)
- Single-file app on GitHub Pages, installable as a PWA
- 3 themes (Maroon Glow / Monochrome / Light)
- Explore (Hot/Rising/New/Random), posting, votes, comments, share permalinks
